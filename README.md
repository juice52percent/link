# 果汁リンク集

静的HTMLのリンク集サイトです。`index.html` をブラウザで開くか、GitHub Pages で公開できます。

## 画像

使用画像は `assets/` に保存しています。

- `assets/hero-art.png`
- `assets/blue-balloon.png`
- `assets/pink-balloon.png`

## 公開

GitHub Pages で公開する場合は、リポジトリの `Settings` → `Pages` から `Deploy from a branch` を選び、`main` / `/ (root)` を指定してください。
